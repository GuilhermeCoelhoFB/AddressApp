v.1 - adressApp:
As mudanças implementadas no código adressApp na versão que se encontra na data de hoje que envio o código, são baiscamente mudanças para aplicação dos padrões de projeto, ou seja, deixando o código mais legível e melhor dividido e de fácil compreensão. Eu busquei criar uma classe abstrata dos métodos de gerenciamento do bd, para que no futuro caso seja preciso mudar o mesmo, não seja tão complexo. Foram criadas algumas classes de genrenciamente de Person, para genrenciar o perfil que vai ser persistido, e uma Util para Person para podermos mudar algumas coisas sem maiores efeitos no código inteiro. Menu de console criado com cases para cada uma das operações e melhorias em alguns trechos originais de código. Pretendi algumas melhorias de tratamento de exceções no código, mas infelizmente algumas das exceções eu não consegui gerenciar, trabalho que será melhorado na próxima versão de AdressApp. Dentro da Main so ficaram as instâncias mais importantes e uma chamada para o menu de escolha de operação.
